# Differences Between Cryo-ET and Single-Particle Analysis (SPA)

| Feature | Cryo-ET | SPA |
|---------|--------|-----|
| Sample Type | Unique cellular specimens | Purified macromolecules |
| Resolution | Lower (~5-10 nm) | Higher (2-4 Å) |
| Structural Context | Preserved | Lost |
